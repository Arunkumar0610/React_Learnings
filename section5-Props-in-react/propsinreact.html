Props In React

Components let you split the UI into independent, resuasable peices, and think about each piece in isolation. Conceptually, components are like JavaScript functions. They accept arbitrary inputs (called "props") and return React elements describing what should appear on the screen

=> props simply means short hand for properties
=> Props- How components talk to each other
=> Props- are a way of passing data from parent to child.
=> Props- Flow downward (unidirectional)
Note: remember that react is said to have unidirectional data flow.
=> Props- Immutable data (read only)
